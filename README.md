# Personal Skills

This repository stores personal Codex skills and helper scripts.

Current skills:

- `skills/bb-browser`: generic `bb-browser` workflow
- `skills/bilibili`: Bilibili-specific browsing helpers built on `bb-browser`
- `skills/latest-tech-research`: time-sensitive technical research via local `grok-search` and primary sources

Conventions:

- Put each skill under `skills/<skill-name>/`
- Keep runnable helpers under `skills/<skill-name>/scripts/`
